DEPLOY INSTRUCTIONS
-------------------
1. Unzip this archive on your computer.
2. You should see all the files and folders listed above sitting next to each other.
3. Upload the ENTIRE contents of this folder (every file and folder, NOT a
   folder-inside-a-folder) into the website root of the hosting account
   (typically /public_html/ or /www/ or /htdocs/).
4. Make sure index.html ends up at the root, e.g. public_html/index.html
   (NOT public_html/Nyumbani-Support-Solutions/index.html).
5. Visit the domain — the homepage should load with all images, styles, and
   navigation working.

QUICK SANITY CHECK BEFORE DEPLOY
--------------------------------
Open index.html locally in any browser by double-clicking it. The page should
appear fully styled with all hero/service/caregiver images visible. If it does,
the package is complete.
